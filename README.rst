My personal homepage and blog
=============================

This is a static site generated by Pelican_.

The custom theme is defined in the ``theme/`` directory.

The following tox commands control generation:

- Generate the static site into the ``output`` directory:

.. code:: console

  tox

- Run the development server (with autoreload on changes):

.. code:: console

  tox -e serve

- Generate pygments light/dark CSS assets:

.. code:: console

  tox -e render-pygments

- Generate in publish mode (into the ``output`` directory):

.. code:: console

  tox -e publish


.. _Pelican: https://getpelican.com
